Installing OpenLogos¶

The source code for OpenLogos can be found on the OpenLogos GitHub repo, as well on SourceForge.

To install you can clone the repo locally and follow the instructions in INSTALL. The database needed to run OpenLogos is available in the same repo in the database/ folder.

There is also Torsten Scheck’s 2006 article (in German).

The OpenLogos Tools are available in a separate OpenLogos-tools GitHub repo, as well as on SourceForge.
